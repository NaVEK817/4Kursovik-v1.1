# S7 Recruitment - Десктопное приложение для рекрутинга

Профессиональное приложение для рекрутинга с современным дизайном в стиле S7 Airlines. Позволяет управлять вакансиями, создавать документы для кандидатов, планировать собеседования, обновлять данные через парсер hh.ru и анализировать кандидатов с помощью ИИ.

## Функциональность

### Основные возможности:
- 🔐 **Авторизация** - вход в систему с проверкой через bcrypt
- 📊 **Главное окно** - таблица всех вакансий из `vacancy_test_file.json`
- 📄 **Оформление документов** - создание DOCX-файлов с информацией о вакансии для кандидата
- 📅 **Расписание собеседований** - календарь с бронированием слотов и отслеживанием создателя
- 🔄 **Обновление данных** - запуск парсера `parcer.py` для загрузки актуальных вакансий с hh.ru

### Новые функции:
- 👥 **Управление пользователями** - добавление, редактирование и удаление пользователей (только для admin)
- 👁️ **Контекстное меню** - правый клик на вакансии для просмотра деталей, создания документа или анализа
- 🤖 **AI-агент** - анализ кандидатов с вычислением рейтинга и рекомендацией лучшего

## Технологии

- Python 3.8+
- PyQt5 - графический интерфейс
- python-docx - создание документов Word
- requests - HTTP-запросы для парсера
- bcrypt - хеширование паролей
