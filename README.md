## Запуск (Python 3)

Используйте **python3**.

### Установка зависимостей

```bash
python3 -m pip install -r requirements.txt
```

### Запуск приложения

```bash
python3 interface.py
```

### Примечания

- При первом запуске появится окно создания первого пользователя, затем окно входа.
- Пользователи сохраняются в `users.json` (пароли хранятся как bcrypt-хэши).
